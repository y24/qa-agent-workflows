# 抽出カテゴリカタログ

## 主要カテゴリ

| カテゴリ | 定義 | 例 |
|---|---|---|
| `feature` | システムが提供する機能、処理、ユースケース | 申請登録、承認、検索、出力 |
| `screen` | 画面、ページ、画面遷移、画面単位の仕様 | 契約登録画面、一覧画面、詳細画面 |
| `ui_item` | 入力項目、表示項目、ボタン、リンク、ラベル | 契約日、保存ボタン、ステータス表示 |
| `report` | 帳票、CSV、PDF、出力ファイル、印刷 | 月次帳票、CSVダウンロード |
| `api` | API、外部IF、リクエスト、レスポンス、連携項目 | POST /contracts、連携ファイル |
| `batch` | 定期実行、夜間処理、ジョブ、再実行 | 月次締めバッチ、連携取込ジョブ |
| `data` | テーブル、マスタ、データ項目、コード、履歴 | 契約マスタ、取引明細、ステータス |
| `business_rule` | 業務上の判断、制約、禁止事項、成立条件 | 締め後は修正不可、承認者は申請者と異なる |

## 判定・制御カテゴリ

| カテゴリ | 定義 | 例 |
|---|---|---|
| `validation` | 入力チェック、形式、必須、桁数、範囲 | 必須、最大10桁、日付形式 |
| `state_transition` | 状態、ステータス、状態遷移、遷移可否 | 未申請→申請中→承認済み |
| `condition_branch` | 条件分岐、判定、処理切替 | 区分Aなら税率X、区分Bなら税率Y |
| `calculation` | 計算、集計、丸め、換算、式 | 税込金額、合計、端数処理 |
| `permission` | 権限、ロール、認可、参照・更新可否 | 管理者のみ削除可 |
| `error_handling` | エラー、例外、失敗時処理、リカバリ | 連携失敗時はエラーログを出力 |

## 補助カテゴリ

| カテゴリ | 定義 | 例 |
|---|---|---|
| `operation` | 運用手順、手作業、運用タイミング | 月末に担当者が手動確認する |
| `non_functional` | 性能、可用性、互換性、セキュリティ、使用性 | 5秒以内に表示、IE非対応 |
| `open_issue` | 未決事項、確認事項、仕様未確定、矛盾 | 承認後修正可否は未定 |

## カテゴリごとのテスト観点例

| カテゴリ | 観点例 |
|---|---|
| `validation` | 有効値、無効値、境界値、未入力 |
| `state_transition` | 許可遷移、禁止遷移、状態別操作可否 |
| `calculation` | 端数、桁数、合計、差分、丸め |
| `permission` | ロール別の参照・登録・更新・削除可否 |
| `api` | 必須項目、型、桁数、異常系、再送 |
| `batch` | 起動条件、再実行、重複、異常終了 |
