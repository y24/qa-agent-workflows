# gap_and_review_report_template.md

## 1. このテンプレートの目的

このテンプレートは、Step 5「不明点・矛盾・不足のレビュー」の成果物である `gap_and_review_report.md` を作成するためのものである。

`gap_and_review_report.md` の目的は、これまでの成果物から、テスト設計に進む前に確認・解消すべきギャップを整理することである。

このテンプレートでは、以下を行う。

- 不明点を整理する
- 資料間・仕様間の矛盾を整理する
- 記載不足を整理する
- 粒度不足を整理する
- 未確定事項を整理する
- 参照元が弱い情報を整理する
- 未処理範囲と影響を整理する
- テスト設計への影響を明示する
- 推奨対応と優先度を付与する

このテンプレートでは、以下を行わない。

- 矛盾を勝手に解消する
- 不足情報を推測で補完する
- 仕様の正解を独断で決める
- テストケースを作成する
- 期待結果を推測で作成する

---

## 2. 使用ルール

このテンプレートを使用する際は、以下を守る。

- 各ギャップに `GAP-ID` を付与する
- ギャップ種別を明示する
- 対象を明示する
- 内容を具体的に記録する
- テスト設計への影響を記録する
- 推奨対応を記録する
- 優先度を付与する
- 関連する `DOC-ID`、`EXT-ID`、`SPEC-ID`、`TDI-ID` を記録する
- 不明点を仕様として補完しない
- 矛盾を独断で解消しない
- 未処理範囲がある場合は、完全版として扱わない
