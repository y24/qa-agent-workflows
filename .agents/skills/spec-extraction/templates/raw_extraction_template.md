# raw_extraction_template.md

## 1. このテンプレートの目的

このテンプレートは、Step 2「事実情報の抽出」の成果物である `raw_extraction.md` を作成するためのものである。

`raw_extraction.md` の目的は、開発ドキュメントに明記されている仕様情報を、原文の根拠付きで抽出し、後続の正規化・統合工程で利用できる形に整理することである。

このテンプレートでは、以下を行う。

- ドキュメントに明記された事実情報を抽出する
- 抽出情報に `EXT-ID` を付与する
- 参照元となる `DOC-ID` と原文位置を記録する
- カテゴリを付与する
- 確度を付与する
- 曖昧・未確定な記述を記録する
- 未処理範囲を明示する

このテンプレートでは、以下を行わない。

- 複数資料の内容を統合する
- 仕様の正誤を判断する
- 不足情報を補完する
- テスト観点へ変換する
- テストケースを作成する

---

## 2. 使用ルール

このテンプレートを使用する際は、以下を守る。

- 原文に明記された内容のみ抽出する
- 推測で仕様を補完しない
- 曖昧な記述を断定表現に変えない
- 抽出情報ごとに `EXT-ID` を付与する
- 参照元 `DOC-ID` を必ず記録する
- 原文位置を可能な限り記録する
- 不明な項目は空欄にせず、`不明`、`未記載`、`要確認` などと記載する
- 複数の仕様が1文に含まれる場合は、必要に応じて分割する
- 後続の正規化・統合は Step 3 に任せる
