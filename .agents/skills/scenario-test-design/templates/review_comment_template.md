# レビューコメントテンプレート

## 1. 目的

このテンプレートは、各stepの成果物に対するユーザーレビュー結果を記録するためのものである。

レビューコメントは、後続stepの入力として使用する。  
AIエージェントは、レビューコメントを参照し、ユーザーが承認・修正・保留・除外した内容を反映して作業する。

## 6. コメント種別の定義

| 種別  | 意味             |
| --- | -------------- |
| 追加  | 成果物に項目を追加する    |
| 修正  | 既存項目の内容を修正する   |
| 削除  | 既存項目を削除候補にする   |
| 統合  | 複数項目を1つにまとめる   |
| 分割  | 1つの項目を複数に分ける   |
| 確認  | 仕様確認・ユーザー判断が必要 |
| 対象外 | 今回の設計対象から外す    |
| 保留  | 現時点では判断しない     |

## 16. AIエージェントへの注意

AIエージェントは、このレビューコメントを参照する際、以下を守ること。

* 承認済み項目は、後続stepの前提として扱う
* 修正が必要な項目は、後続stepへ進む前に反映する
* 対象外項目は、後続stepで勝手に復活させない
* 保留項目は、断定せず保留として扱う
* 確認事項への回答は、該当する成果物と後続stepへ反映する
* レビューコメントと既存成果物が矛盾する場合は、レビューコメントを優先する
* 判断に迷う場合は、確認事項として扱う
* ユーザーの明示指示なしに、対象外項目を再度シナリオ化・テストケース化しない
