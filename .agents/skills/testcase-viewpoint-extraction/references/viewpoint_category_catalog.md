# 観点カテゴリカタログ

## 機能・条件系

| カテゴリ | 典型的な観点 |
|---|---|
| 入力値・バリデーション | 必須、形式、桁数、範囲、無効値、未入力 |
| 境界値 | 最大、最小、0、NULL、期間開始・終了、月末・年度末 |
| 状態・ステータス | 状態別操作可否、状態遷移、禁止遷移、遷移後表示 |
| 条件分岐 | 複数条件、フラグ、区分、判定結果、処理切替 |
| 権限・ロール | ロール別表示、操作可否、承認、参照範囲 |

## 業務・データ系

| カテゴリ | 典型的な観点 |
|---|---|
| 業務フロー | 一連の業務操作、前後工程、承認、締め、取消 |
| データ整合性 | 重複、欠落、更新反映、履歴、マスタ整合 |
| 計算・集計 | 合計、差分、丸め、税率、集計対象 |
| 回帰 | 既存仕様維持、過去不具合再発、共通部品影響 |

## 出力・連携・非機能系

| カテゴリ | 典型的な観点 |
|---|---|
| 表示・UI | 表示内容、表示制御、メッセージ、初期値 |
| 帳票・出力 | CSV、PDF、帳票項目、出力条件、ダウンロード |
| 外部連携 | API、ファイル、送受信、再送、異常系 |
| エラー・例外 | エラー表示、ロールバック、リカバリ、再実行 |
| 性能・大量データ | 件数、応答時間、タイムアウト、大量登録 |
| 操作性・ユーザビリティ | 誤操作防止、案内、入力補助、導線 |
| バッチ・非同期処理 | ジョブ起動、完了確認、非同期結果、再実行 |
| 環境・設定 | ブラウザ、OS、設定値、構成差 |

## 補助カテゴリ

| カテゴリ | 使い方 |
|---|---|
| 分類保留 | 根拠不足、複数カテゴリ候補、レビュー待ち |
| その他 | 既存カテゴリに当てはまらないが、根拠TCがある場合のみ |
