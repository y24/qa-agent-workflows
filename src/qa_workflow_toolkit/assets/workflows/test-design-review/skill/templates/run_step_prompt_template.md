# Run Step Prompt Template

```markdown
$test-design-review の Step <番号> を実行してください。

## 対象

- 対象システム・機能:
- レビュー対象成果物:
- レビュー範囲:

## 入力

- <レビュー対象ファイルまたは前Step成果物>
- <根拠資料>

## 出力先

- <指定がなければ shared/output_location_policy.md に従う>

## 注意事項

- 根拠資料にない仕様、業務ルール、期待結果を創作しない。
- 不足と断定できない事項は、未確認事項として明示する。
- テストケースを勝手に大量生成しない。
- Step完了後は次Stepへ進まず、レビュー待ちで停止する。
```
