# 観点カテゴリ seed

## 目的

抽象観点カタログで使用する観点カテゴリの初期セットを定義する。

カテゴリは整理の補助であり、主な確認対象に最も近いカテゴリを1つ選ぶ。迷う場合は分類保留にする。

## カテゴリ一覧

| カテゴリ | 主な確認対象 |
|---|---|
| 入力値・バリデーション | 必須、形式、桁数、範囲、有効値・無効値 |
| 境界値 | 最大、最小、0、NULL、日付境界、期間境界 |
| 状態・ステータス | 状態遷移、状態別動作、禁止遷移 |
| 条件分岐 | 条件組合せ、判定、処理切替 |
| 権限・ロール | 参照、登録、更新、削除、承認の可否 |
| 業務フロー | 複数操作の流れ、前後工程、業務目的 |
| データ整合性 | 重複、欠落、更新反映、履歴、マスタ整合 |
| 計算・集計 | 算出式、合計、丸め、差分 |
| 表示・UI | 表示内容、表示制御、メッセージ |
| 帳票・出力 | CSV、PDF、帳票、ダウンロード |
| 外部連携 | API、ファイル連携、送受信、再送 |
| エラー・例外 | エラー処理、例外、リカバリ |
| 回帰 | 既存機能影響、過去不具合再発 |
| 性能・大量データ | 応答時間、件数、負荷、タイムアウト |
| 操作性・ユーザビリティ | 分かりやすさ、誤操作防止、導線 |
| バッチ・非同期処理 | ジョブ、スケジュール、非同期結果 |
| 環境・設定 | ブラウザ、OS、設定、構成差 |
| 分類保留 | 根拠不足、複数候補、レビュー待ち |
| その他 | 既存カテゴリに当てはまらないが根拠がある観点 |
