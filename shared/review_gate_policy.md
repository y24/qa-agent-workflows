# Review Gate Policy

## Default Gate

複数ステップの workflow は、各主要ステップの完了時に停止し、ユーザーのレビューまたは継続指示を待つ。

## Continue Conditions

次に進めるのは、以下のいずれかの場合に限る。

- ユーザーが明示的に次ステップへの継続を指示した。
- ユーザーが最初に「最後まで実行」などの明示的な一括実行を依頼した。
- 対象 skill の `SKILL.md` または step guide が例外条件を定義している。

## Gate Output

レビュー待ちでは、以下を簡潔に示す。

- 作成した成果物
- 主な内容
- 重要な確認事項
- 次に進む場合のステップ名

## Corrections

前段成果物が修正された場合は、後続成果物への影響範囲を示す。必要なら影響を受けるステップを再実行する。
